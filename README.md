# Soongsil Campus Map <br/>
### 📖 < 프로젝트 소개 > <br/>
"Soongsil Campus Map"은 숭실대학교 신입생, 동작구에 새로 전입 온 재학생들을 위한 지역 정보제공 서비스입니다. 숭실대학교 학생들, 특히 학교가 처음인 사람들을 대상으로 교내의 시설 안내 및 
유익한 정보들을 제공합니다. 교내의 유용한 정보들은 직접 다녀보거나 아는 선배에게서 듣는 수밖에 없습니다. 하지만 갓 입학한 신입생들은 모든것이 새롭기만 하고 특히나 코로나 이후 입학한 학생들은 아는 선배도 적기에 막막하기만 합니다. 그렇기에 건물 정보도 볼 수 있고 생활 팁도 찾아볼 수 있다면 생활의 불편함을 개선하는데 큰 도움이 될 것입니다.

### 📖 < 기능 설명 > <br/><br/>
## 🎵 1. 지도 시스템 <br/>
학교에 처음 들어오게 되면 길을 헤메는 경우가 많습니다. 이럴 경우 지도 앱을 켜면 된다고 생각할 수도 있지만 일반적인 지도는 피라미드나 생협같이 교내에 특화된 장소는 알려주지 않는다는 문제점이 있습니다. 그렇기에 숭실 캠퍼스맵에서는 학교 생활에 도움이 되는 장소들에도 마크를 배치해두어 기존 지도들과 차별점을 두었습니다. 마크를 클릭할 경우 해당 시설에 대한 팁을 제공해주기 때문에 팀플할 장소를 찾기 등에 유용하게 쓰일 수 있습니다. 시설별로 메모 기능도 제공되기 때문에 "공강떄 여기 있으니까 편하더라" 등 자신만의 팁도 기록할 수 있습니다. <br/><br/>

## 🎵 2. 게시판 시스템 <br/>
이미 에브리타임 등과 같은 어플에서 게시판을 통한 질문이 원활하게 일어납니다. 그러나 사람들과의 소통에 주 목적을 둔 게시판의 경우 유실되는 정보가 많다는 문제점이 있습니다. 매년 신입생들은 비슷한 내용의 질문을 하지만 누군가가 대답해줘도 금방 다른 글들에 묻히고 똑같은 질문이 다시 올라옵니다. 가끔씩 잘 정리된 글이 올라오더라도 금새 사라지고 다음 해에도 똑같은 질문이 올라오는 현상이 발생합니다. 이러한 과정에서 놓치는 정보들도 생기고 대답해주는 사람이 없으면 정보를 구하지 못하기 때문에 정작 필요할땐 도움이 되지 못하기도 합니다.그렇기에 숭실 캠퍼스맵에서는 유용한 정보들에 특화된 게시판을 따로 두어 시간이 지날수록 유익한 정보가 쌓여가도록 만들었습니다. 또한 비슷한 주제의 팁은 같은 카테고리에 묶어두어 한눈에 연관된 팁을 볼 수 있도록 하였습니다. <br/><br/>

## 🎵 3. 유저 시스템 <br/>
숭실 캠퍼스맵에서는 숭실대학교의 공식 색상들을 사용합니다.<br/>
<img src = "https://user-images.githubusercontent.com/47634717/121775524-34cf6380-cbc3-11eb-92b3-799d21d71b5b.png" width="50%"/> <br/>
색감 뿐만 아니라 학번으로 로그인하는 등의 방식을 사용하여 기존의 공식 사이트들(스마트 캠퍼스, 유세인트, 슈팅 등)과 통일감을 주었습니다. 이처럼 유저 관리 기능이 있긴 하지만 기본적으로 번거로운 로그인이나 회원가입 없이도 핵심 기능들의 사용이 가능합니다. 금만 더 늦으면 지각할 것 같은데 지도를 보기 위해선 학생증 인증도 하고 회원가입도 해야한다고 한다면 사용자 수는 급격하게 감소할 것입니다. 그렇기에 켜기만 한다면 지도보기, 메모하기, 게시판 글 읽기등이 가능하도록 하여 진입장벽을 대폭 낮추었습니다.
